HTML5 is an innovative and updated programming language

5 contains about 100 specifications related to the next generation of Web technologies. The name HTML5 brings them together together. In practice, HTML5 brings together innovations in HTML, CSS3 and JavaScript designed to allow developers to build the next generation of web applications.

history:

*
HTML (Languag Markup Text Hyper) (created in 1991 as a way to transfer text, images and links to other web pages.

*
In 1994-1995 the world advanced to the HTML2 standard which contained important features such as tables, uploading files and such.

*
In 1996, the concepts CSS1 (style sheet) and JavaScript (code snippets for html pages) were accepted the HTML capabilities from a page capable of displaying only static information to a page with dynamic information.

*
In 1997, the standard advanced to HTML4 with Frames and other capabilities.

*
In 1998 - CSS progressed to version 2, with new design features.

*
In 2000 - the standard changed to XHTML and caused the HTML developers to write a more hardened and standardized code.

*
In 2005, the concept of AJAX (asynchronous access from the client to the server) entered the standard.

*
Starting in 2009, they started talking about HTML5.
